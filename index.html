Capabilities

What the System Can Do

Four portals, one platform — each designed for a specific role.

badge Role-Based Portals

  • check_circle Executive Path — Travel management: trip planning, itinerary, expenses, reports, settings, and guide.
  • check_circle Executive Task — Task-focused dashboard with planner, daily log, and milestone tracking.
  • check_circle Executive IT — IT service desk: service reports, maintenance checklists, asset inventory, Kanban task board, planner, accomplishments, support tickets, and system monitoring.
  • check_circle Administrator — Central control panel: manage users, view all tasks/plans/trips/logs, data backup & restore, and global search.

storage Data & Privacy

  • check_circle Client-Side Storage — All data stays in your browser's local storage. No servers, no cloud, no third-party access.
  • check_circle Backup & Restore — Export all data as JSON. Import to restore or migrate between devices.
  • check_circle Isolated by Design — Each user's data is isolated in their own localStorage namespace. Admin accounts can view all data across users.
